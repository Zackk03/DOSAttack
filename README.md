# Requisitos
```
sudo apt-get install python3
```
```
sudo yum install python3
```

# Instalación
```
git clone https://github.com/Zackk03/DOSAttack.git
cd DOSAttack
```

# Ejecución
```
./DOSAttack.py
```
```
sudo python3 DOSAttack.py
```
